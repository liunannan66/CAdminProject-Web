# 付款条件管理 — 前后端联调说明

## 后端接口

| 接口 | 方法 | 权限码 |
|------|------|--------|
| `/api/PaymentTerm/PageList` | GET | `api:PaymentTerm:PageList` |
| `/api/PaymentTerm/ListEnabled` | GET | `api:PaymentTerm:ListEnabled` |
| `/api/PaymentTerm/Add` | POST | `api:PaymentTerm:Add` |
| `/api/PaymentTerm/Update` | POST | `api:PaymentTerm:Update` |
| `/api/PaymentTerm/Delete/{id}` | GET | `api:PaymentTerm:Delete` |

`prepay` `progress` `delivery` `acceptance` `warranty` `monthly`

## 业务规则

- 条件名称唯一（未删除记录）
- 支付比例 0～100
- `ListEnabled` 供合同/立项收款条件选择（后续迭代）

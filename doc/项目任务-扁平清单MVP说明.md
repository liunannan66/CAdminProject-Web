# 项目任务（扁平清单）MVP 说明

## 方案确认

- **任务结构**：扁平任务清单（非 WBS 树）
- **移动端**：H5 页面 `/m/task`（非微信小程序）
- **入口**：必须从 **已立项**（`status = approved`）项目进入

## 功能范围

| 端 | 功能 |
|----|------|
| PC | 项目立项列表 →「任务清单」→ 增删改查、分配责任人、进度/状态 |
| H5 | `/m/task` 我的任务、筛选、更新进度与状态 |

## 部署步骤

1. 执行 SQL：
   - `doc/sql/add-pi_project_task.sqlserver.sql`
   - `doc/sql/add-project-task-permission-role49.sqlserver.sql`
2. 重启 `MalusAdmin.WebApi`
3. 重新登录 PC 端（刷新按钮权限）

## 使用路径

1. 将项目立项状态设为 **已立项**
2. 项目立项列表 → 点击 **任务清单**
3. 新增任务并指定责任人
4. 责任人手机浏览器访问：`{前端域名}/m/task`（需先登录系统）

## 后续可扩展

- 任务池模板批量导入
- 消息通知（分配/逾期提醒）
- 数据看板
- 升级为 WBS 四级树

# 报价器管理 — 前后端联调说明

## 后端接口

| 接口 | 方法 | 权限码 |
|------|------|--------|
| `/api/Quoter/PageList` | GET | `api:Quoter:PageList` |
| `/api/Quoter/ListModuleNames` | GET | `api:Quoter:ListModuleNames` |
| `/api/Quoter/ListEnabled` | GET | `api:Quoter:ListEnabled` |
| `/api/Quoter/Add` | POST | `api:Quoter:Add` |
| `/api/Quoter/Update` | POST | `api:Quoter:Update` |
| `/api/Quoter/Delete/{id}` | GET | `api:Quoter:Delete` |

## 数据库脚本

1. `doc/sql/create-bd_quoter.sqlserver.sql`
2. `doc/sql/add-quoter-button-permission.sqlserver.sql`（默认 RoleId=6）

执行权限脚本后请重新登录。

## 前端文件

- `src/service/api/basedata.ts` — 报价器 CRUD
- `src/constants/basedata-quoter.ts`
- `src/views/basedata/quoter/index.vue`
- `src/views/basedata/quoter/modules/quoter-operate-dialog.vue`

## 业务规则

- 模块名称 + 子模块名称在未删除记录中唯一
- 单价可选（可为 null）
- `ListEnabled` 仅返回启用项，供立项「实施内容」后续选择
